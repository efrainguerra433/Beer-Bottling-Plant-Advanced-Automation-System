This folder contains documentation related to the SCADA system developed
using Citect SCADA.

The SCADA application was used to emulate field sensors and actuators
and to provide operator supervision of the beer bottling process.

## SCADA Features
- Real-time visualization of the bottling line
- Emulated sensors and actuators
- Trend visualization
- Operator control interface

Due to the proprietary nature of Citect SCADA, only screenshots and
documentation are provided.
